# 🎮 Just Divide Game

A React-based math puzzle game where users drag and drop numbers to merge them based on division rules.

---

## 🚀 Live Demo
👉 https://just-divide-game-t5fa.vercel.app

---

## 📌 Features
- 🎯 Drag and Drop gameplay
- 🔄 Undo functionality
- 🧮 Score tracking system
- 📈 Level progression
- 💾 Best score saved using localStorage
- 🗑️ Keep & Trash system

---

## 🛠 Tech Stack
- React.js
- JavaScript (ES6)
- HTML5 & CSS3

---

## ⚙️ How to Run Locally

```bash
git clone https://github.com/NishuBansal1110/just-divide-game.git
cd just-divide-game
